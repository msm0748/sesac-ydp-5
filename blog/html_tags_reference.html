HTML 블록 레벨 요소 태그

HTML 블록 레벨 요소 태그에 대한 설명
태그명 태그의 의미 및 특징
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>
  • HTML 문서의 제목(Heading)을 정의할 때 사용
  • 하나의 HTML 문서에는 하나의 h1 태그를 권장
  • h1 ~ h6 태그는 텍스트와 인라인 요소를 포함할 수 있지만 블록 레벨 요소는 포함할 수 없음
  • 웹 검색 엔진이 제일 먼저 검색하는 태그 !
  • 스크린 리더(음성낭독프로그램)에서는 제목의 수준으로 바로 이동할 수 있기 때문에 h1 -> h2 -> h3 -> h4 -> h5 -> h6 순으로 정의하는 것이 좋으며, 중간에 제목의 수준이 빠지지 않도록 주의
  • 보통 h2 태그부터는 같은 레벨의 제목 태그를 여러번 정의할 수 있음
<p></p>
  • HTML 문서에서 단락(paragraph)을 정의할 때 사용
  • 텍스트와 인라인 요소를 포함할 수 있지만 블록 레벨 요소는 포함할 수 없음
  • 텍스트의 행을 바꾸려면 br 태그를 사용
<address></address>
  • HTML 문서의 소유자나 조직에 대한 연락처 정보를 정의할 때 사용
  • 텍스트와 인라인 요소를 포함할 수 있고 h1 ~ h6, section, article, aside, nav, header, footer와 같은 일부 블록 레벨 요소들은 포함할 수 없음
  • address 태그는 보통 footer 태그 내에서 많이 정의
<ul></ul>
  • Order List의 약자로 순서 목록을 정의할 때 사용
  • 자식 요소로 반드시 li 태그를 정의해 주어야 하며, li 태그 이외의 다른 태그는 자식 요소로 올 수 없음
<ul></ul>
  • Unorder List의 약자로 비순서 목록을 정의할 때 사용
  • 자식 요소로 반드시 li 태그를 정의해 주어야 하며, li 태그 이외의 다른 태그는 자식 요소로 올 수 없음
<li></li>
  • list items로 항목을 정의할 때 사용 (ul, ul 태그의 자식 요소)
  • 텍스트, 인라인 요소, 블록 레벨 요소를 모두 포함할 수 있음
<div></div>
  • 요소들을 그룹으로 정의할 때 사용
    요소들을 그룹으로 묶어주는 이유는 CSS를 손쉽게 적용하기 위한 것이고, 태그 자체는 아무런 의미를 갖고 있지 않음
  • 텍스트, 인라인 요소, 블록 레벨 요소를 모두 포함할 수 있음

HTML 구조 관련 요소 태그 (블록 레벨 요소)

HTML 구조 관련 요소 태그에 대한 설명 <블록 레벨 요소>
태그명 태그의 의미 및 특징
<header></header>
  • HTML 문서의 헤더 영역을 의미하는 태그로 제목이나 내비게이션, 검색 등의 내용들을 포함할 수 있음
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있지만 header, main, footer 태그는 포함할 수 없음
<nav></nav>
  • HTML 문서의 메인 메뉴나 목차 등을 정의할 때 사용
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있음
<main></main>
  • HTML 문서의 콘텐츠 영역을 정의할 때 사용
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있지만 header, footer 태그는 포함할 수 없음
<section></section>
  • HTML 문서에서 맥락이 같은 요소들을 주제별로 그룹화해주는 태그이며 섹션 주제에 대한 제목 요소(h2 ~ h6)를 포함하는 것이 바람직함
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있음
<article></article>
  • HTML 문서 내에서 독립적으로 배포 또는 재사용이 가능한 게시물, 뉴스 기사, 블로그 포스팅 등을 의미하는 태그이며 제목 요소(h2 ~ h6)를 포함하는 것이 바람직함
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있음
<aside></aside>
  • 메인 콘텐츠와 직접적으로 관련이 없는 영역을 의미하는 태그이며 HTML 문서의 오른쪽이나 왼쪽의 사이드 메뉴나 광고 등의 영역으로 사용
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있음
<footer></footer>
  • HTML 문서의 푸터 영역을 의미하는 태그로 섹션 작성자나 저작권에 대한 정보, 관련된 문서의 링크를 포함할 수 있음
  • 텍스트, 인라인 요소, 블록 레벨 요소를 포함할 수 있지만 header, main, footer 태그는 포함할 수 없음
